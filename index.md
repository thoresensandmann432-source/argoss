# ARGOS Universal OS — Documentation

Добро пожаловать в документацию ARGOS.

Здесь три основных направления:

- User Guide — как установить, запустить и использовать систему в повседневной работе.
- Developer Guide — как писать новые навыки (плагины), подключать их и отлаживать.
- Philosophy — концепция проекта: цифровое бессмертие, квантовое управление и лор ARGOS.

## Быстрый старт документации

```bash
pip install -r docs/requirements-docs.txt
mkdocs serve
```

После запуска сайт будет доступен локально на http://127.0.0.1:8000.
